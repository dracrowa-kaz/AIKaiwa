# AIKaiwa(AI会話)
====

Overview

## Description
You can talk to AI chat bot.  
Your text will send to AI API server , and you can receive text from AI bot.  
Please enjoy talking to AI.  

## Demo
![Imgur](http://i.imgur.com/sxMHueX.gifv)

## Requirement
Swift 3   
Xcode 8  

## Install
1. Pod install.  
2. Add docomo-api-key.   
3. Build.
